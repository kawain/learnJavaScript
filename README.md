## ライブラリ沢山

### npm 

https://www.npmjs.com/

世界中の1,100万人を超える開発者に依存されているnpmは、JavaScript開発をエレガントで生産的で安全なものにすることに取り組んでいます。
無料NPMレジストリは、100万本以上のパッケージ、世界最大のソフトウェアレジストリとJavaScriptのコード共有の中心地となっています。

### 利用人口が多い

### しかしJS自体の変化が速いので
